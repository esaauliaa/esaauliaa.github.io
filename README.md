# Personal Portfolio Website - Esa Aulia

Website portofolio pribadi yang responsif dan modern, dibuat menggunakan HTML, CSS, dan JavaScript. Website ini digunakan untuk menampilkan profil, keahlian, layanan, serta proyek yang pernah dikerjakan dengan tampilan yang bersih dan estetis.

---

## Fitur

- Desain antarmuka modern dan responsif
- Navigasi dengan efek smooth scrolling
- Highlight aktif pada navbar
- Animasi saat scroll (fade dan slide)

- Halaman Home sebagai tampilan utama (hero section)

- Halaman About Me:
  - Informasi profil diri
  - Latar belakang pendidikan

- Halaman Service:
  - Menampilkan layanan yang ditawarkan:
    - Web Development
    - UI/UX Design
    - System Analysis
    - Graphic Design

- Halaman Portfolio:
  - Menampilkan proyek yang pernah dikerjakan

- Halaman Contact:
  - Link interaktif yang dapat diklik:
    - Email
    - WhatsApp
    - LinkedIn
    - Instagram
---

## Teknologi yang Digunakan

- HTML5
- CSS3 (Flexbox dan Responsive Design)
- JavaScript (DOM Manipulation dan Animation)
- Font Awesome (Icon)
- Google Fonts
